Avaliações:
Trabalhos/Provas

Instalar no ubuntu 14.04 lts:
O python já vem instalado em distribuições linux
Instalar PIP: 
- sudo apt-get -y install python-pip
Instalar Flask:
- sudo pip install Flask
Instalar suporte Python com PostgreSQL:
- sudo apt-get install python-psycopg2

Instalar no windows:
Instalar Python
- https://www.python.org/downloads/
- Habilitar classpath (o único "x" vermelho que não está marcado na instalação)
- Adicionar nas variáveis de ambiente
Instalar PIP: 
- Baixar o arquivo get-pip.py
- Site: https://pip.pypa.io/en/stable/installing/
- E executá-lo: python get-pip.py
Instalar Flask:
- pip install Flask
Link de apoio: http://matthewhorne.me/how-to-install-python-and-pip-on-windows-10/
